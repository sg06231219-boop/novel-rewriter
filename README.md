# 小说翻改工具 v2.0

一键翻改小说：名称替换 + AI 句式改写，剧情不变，焕然一新。

## 功能

- 🔄 **批量名称替换** — 角色名、地名、组织名一键替换，长名称优先匹配
- 🧠 **智能提取名称** — 自动从原文识别人名/地名/门派名，不用手动输入
- ✨ **AI 句式改写** — 接入智谱 GLM-4-flash，轻微/中等/大幅三档改写强度
- 👁️ **左右对比预览** — 原文/改写文并排对照
- 📊 **替换统计** — 实时显示替换次数
- 💾 **下载结果** — 一键导出 TXT

## 使用

```bash
pip install -r requirements.txt
python app.py
# 访问 http://localhost:8000
```

## AI 改写

需要智谱 AI API Key（免费注册 https://open.bigmodel.cn 获取），在页面右上角 🔑 设置。
